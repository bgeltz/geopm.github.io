NAME

geopm_region_id_c - application profiling insight interfaces

SYNOPSIS

#include <geopm_region_id.h>

Link with -lgeopm

int geopm_region_id_is_mpi(

uint64_t region_id);

int geopm_region_id_is_unmarked(

uint64_t region_id);

int geopm_region_id_hint_is_equal(

uint64_t hint_type,
uint64_t region_id);

uint64_t geopm_region_id_hint(

uint64_t region_id);

uint64_t geopm_region_id_hash(

uint64_t region_id);

DESCRIPTION

The functions described here are intended to provide insight with regard to the region being executed by a particular rank as served by the PlatformIO REGION_ID# signal. For more details regarding PlatformIO and signals please see geopm(7).

geopm_region_id_is_mpi()

returns 1 if region_id is currently executing an MPI operation, otherwise returns 0.

geopm_region_id_is_unmarked()

returns 1 if region_id is currently executing unmarked code, otherwise returns 0.

geopm_region_id_hint_is_equal()

returns 1 if region_id is associated with hint_type, otherwise returns 0.

geopm_region_id_hint()

returns hint type that is associated with region_id.

geopm_region_id_hash()

returns region_id dissociated from hints.
