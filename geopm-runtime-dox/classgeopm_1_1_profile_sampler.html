◆ cpu_rank()

virtual std::vector<int> geopm::ProfileSampler::cpu_rank ( void  ) const
pure virtual

Retrieve a vector to the affinities of all application ranks.

Return vector is sized to number of Linux online CPUs in the system. Each element of the vector is indexed by the Linux CPU ID, and the value assigned is the MPI rank running on the CPU (or -1 if no rank has been affinitized).

Returns
Vector to be filled with the MPI rank for each Linux CPU, set to -1 if no MPI rank is affinitized.
