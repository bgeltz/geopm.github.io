Guide for Service UsersÔÉÅ

This guide describes why and how to use the GEOPM Service, which provides a userspace interface to access hardware telemetry and settings. The guide includes sections that describe how to build and install this software component of the GEOPM project, how to configure this service as a system administrator, and how to interact with the service as an unprivileged user.
